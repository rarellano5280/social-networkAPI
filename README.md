  ## Description
This is an API soical network app that that will allow for the creation of new users. The user can then create thoughts and also add friends to their freinds list. The user can also share thoughts and their firends can create reactions to the users thoughts. 

  ## Installation
To install this app, one will need to clone my repo and. The user wikk need to have node.js and mongoDB installed on thier computer. Once your environment is set up the user will need to run `npm i`

  ## Usage
Once you've installed all dependencies you can run `npm start` to start up the application. Once the server has been run you will have a new mongoDB database. You can view this database in Compass once you refresh.
